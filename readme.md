
# Lesson 3: Lists

## What we will learn in thie lesson?

- Unordered list.
- Ordered list.
- Parent-children relationship of element.
- Nested lists.

## HTML elements

Unordered list:
```html
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
```

Ordered list:
```html
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
```

Nested lists:
```html
<ol>
    <li>Item 1</li>
    <li>Item 2
        <ul>
            <li>Item 2.1</li>
            <li>Item 2.2</li>
        </ul>
    </li>
    <li>Item 3</li>
</ol>
```
